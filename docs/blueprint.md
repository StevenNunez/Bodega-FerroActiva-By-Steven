# **App Name**: Control de Bodega

## Core Features:

- Login con Roles: Autenticación de usuarios con roles de Administrador, Supervisor y Trabajador.
- Escaneo QR: Permite escanear códigos QR para identificar trabajadores y herramientas.
- Entrega y Devolución de Herramientas: Registra la entrega y devolución de herramientas mediante el escaneo de códigos QR.
- Visualización de Stock Disponible: Los supervisores pueden ver el stock disponible de materiales antes de realizar una solicitud.
- Generación de Solicitudes de Materiales: Los supervisores pueden generar solicitudes de materiales asociadas a un área o proyecto.
- Alertas de Solicitudes: El administrador recibe alertas visuales cuando un supervisor realiza una solicitud de materiales.
- Generar nombres para nuevas herramientas: El administrador puede utilizar un asistente de IA para generar nombres creativos y descriptivos para las herramientas recién ingresadas. The LLM uses a tool to look up the name of similar tools to decide whether to include information in the generated name

## Style Guidelines:

- Color primario: Azul (#29ABE2) para reflejar profesionalismo y confianza.
- Color de fondo: Gris claro (#F0F0F0) para facilitar la lectura y reducir la fatiga visual.
- Color de acento: Naranja (#FF9800) para resaltar elementos importantes como botones de acción y alertas.
- Fuente para cuerpo y títulos: 'Inter', una fuente sans-serif para una apariencia moderna y limpia.
- Utilizar íconos claros y reconocibles para representar las diferentes herramientas y materiales.
- Diseño limpio y organizado para facilitar la navegación y la visualización de la información.
- Animaciones sutiles para proporcionar retroalimentación visual al usuario al interactuar con la aplicación.
